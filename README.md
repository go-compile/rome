# Rome

The Elliptic and Edward Curve cryptography library built for multi-curve use.

# Curves

- nist P-521
- nist P-256
- nist P-224

More to come...

# Features

- Generate key
- Export (Public, Private) PEM **and** ASN.1 DER bytes
- Import (Public, Private) PEM **and** ASN.1 DER bytes
- Sign (ASN.1 format)
- Verify
